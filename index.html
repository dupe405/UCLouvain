 Principe d'authentification


Afin de garantir que les documents émis par l'UCLouvain puissent être distingués de faux, nous fournissons sur ceux qui sont les plus sensibles un QRCODE de validation.

Ce QRCODE contient à l'intérieur toutes les informations présentes sur cette page, et ce, chiffrées de façon cryptographique.

Seul un QRCODE authentique et non modifié peut ainsi être déchiffré et affiché sur cette page sans problème. Afin de garantir la meilleure chaîne d'authentification possible, merci de suivre ces quelques conseils :


  1. Scanner le QRCODE présent sur le document pour être dirigé vers le site de l'UCLouvain
  2. Vérifier que vous êtes bien sur le véritable site https://dupe.uclouvain.be
  3. Confirmer que le programme affiche un statut VERT () ou du moins BLEU ()
  4. En cas de statut ORANGE (), merci de tenir compte des avertissements affichés
  5. Si le statut est ROUGE () cela signifie très certainement que le QRCODE est altéré et/ou non authentique
  6. Comparer les éléments affichés sur cette page avec ceux présents dans le document afin de garantir que le texte du document n'a pas été modifié
